Ship it

Host your portfolio free on GitHub Pages

Each template is a single index.html file — no build step, no server. GitHub Pages will serve it for free at username.github.io/repo-name.

Step 1
Download your template
Open a template and click the ↓ Download button in the toolbar. You'll get a single .html file with everything included.
Step 2
Fill in your details
Open the file in any text editor. Find the const P = { … } block near the top and replace the placeholder content with your own name, bio, skills, and projects.
Step 3
Create a GitHub repo
Go to github.com, sign in, and create a new repository. Name it anything — e.g. portfolio. Leave it public and don't initialise it with any files.
Step 4
Upload your file
Inside your new repo, click Add file → Upload files. Upload your .html file and rename it to index.html before committing.
Step 5
Enable GitHub Pages
Go to your repo's Settings → Pages. Under Branch, select main and set the folder to / (root). Click Save.
Step 6
Your site is live
After about a minute, your portfolio will be live at username.github.io/portfolio. GitHub Pages automatically redeploys whenever you push a change.

Tip: Want it at username.github.io with no path? Name your repository username.github.io exactly (using your real GitHub username). GitHub will automatically serve it at the root of your domain.

Share It

Get your portfolio in front of people

A portfolio no one sees is just a folder. Put your link everywhere.

💼
LinkedIn
Add your portfolio URL to your LinkedIn profile under Contact info → Website. Also post about it — a short "I just launched my portfolio" post gets more eyes than you'd expect.
🐙
GitHub Profile
Pin your portfolio repo on your GitHub profile and add the live URL to the repo's About section. Recruiters who find you on GitHub will see it immediately.
📄
Your Résumé
Add your portfolio URL next to your name at the top of your résumé. Keep the link short — use your username.github.io address or a free custom domain from Namecheap or Cloudflare.
✉️
Email Signature
Add your portfolio link to your email signature. Every message you send is a passive introduction — especially useful when cold-emailing recruiters or collaborators.
🐦
Social Media
Post your link on X/Twitter, Bluesky, or wherever your community lives. Add it to your bio too. Communities like #buildinpublic and #100DaysOfCode are especially receptive.
🌐
Custom Domain
For a more professional look, point a custom domain at your GitHub Pages site. Buy a domain, then add it under repo Settings → Pages → Custom domain. GitHub provides free HTTPS automatically.
